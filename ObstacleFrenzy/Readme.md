Obstacle Frenzy: Gravity-Shift Endless Runner

Obstacle Frenzy is a browser-based endless runner that introduces a high-speed "Gravity Flip" mechanic.
Players must navigate obstacles by toggling between the floor and ceiling in real-time. 
This project explores the fundamentals of 2D Physics and Asynchronous Game Loops in JavaScript.
